# 📅 活动数据添加说明

**日期**: 2026-03-26

---

## 📋 活动方案

由于当前 HAP 应用中没有专门的"活动"工作表，我们使用现有的 **posts（照片/帖子）** 工作表来存储活动数据。

### 字段映射

| posts 表字段 | 活动数据 |
|------------|---------|
| title | 活动名称 |
| content | 活动描述 |
| category | 活动分类（固定为"活动"） |
| task_type | 活动类型（摄影/娱乐/学习/运动/美食） |
| bounty_amount | 奖金/奖品价值 |
| images | 活动图片 |

## 🚀 添加方法

### 方法 1: 运行脚本（推荐）

在微信开发者工具控制台运行：

```javascript
const api = require('./utils/api.js');
// 复制 add-activity-data.js 的内容并运行
```

### 方法 2: 手动添加

1. 打开 HAP 应用
2. 进入「照片/帖子」工作表
3. 点击「新建记录」
4. 填写活动信息
5. 保存

## 🎨 前端展示建议

### 活动列表页

可以创建一个新页面 `pages/activities/activities.js`：

```javascript
// 筛选活动
const activities = posts.filter(post => post.category === '活动');
```

### 活动卡片

显示活动信息：
- 📅 活动名称
- 🏷️ 活动类型
- 💰 奖金（如果有）
