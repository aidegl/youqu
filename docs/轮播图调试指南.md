### 步骤 3: 分析输出结果

#### ✅ 正常情况

看到以下日志表示成功：
```
[API.getBanners] success: true
[API.getBanners] rows 数量：2
[轮播图] ✅ 加载成功，数据：[...]
```

**检查点**：
- `success: true` - API 调用成功
- `rows 数量 > 0` - HAP 中有数据
- `✅ 加载成功` - 数据已设置到页面

#### ❌ 无数据

如果看到：
```
[API.getBanners] success: true
[API.getBanners] rows 数量：0
[轮播图] ❌ 加载失败或无数据
```

**原因**：HAP 中没有 `是否启用=是` 的记录

**解决方案**：
1. 打开明道云 HAP
2. 进入「轮播图」工作表
3. 编辑记录，设置 `是否启用 = 是`
4. 保存

#### ❌ API 失败

如果看到：
```
[API.getBanners] success: false
[轮播图] 错误信息：xxx
```

**可能原因**：
1. HAP-Appkey 或 HAP-Sign 错误
2. 网络问题
3. HAP 接口权限问题

**解决方案**：
1. 检查 `utils/api.js` 中的 `HAP_CONFIG`
2. 确认 Appkey 和 Sign 正确
3. 检查网络连接

### 步骤 4: 手动测试 API

在控制台手动运行：

```javascript
const api = require('./utils/api.js');

console.log('=== 手动测试轮播图 API ===\n');

api.getBanners().then(res => {
  console.log('API 响应:', res);
  
  if (res.success) {
    console.log('✅ 成功！轮播图数量:', res.data.length);
    res.data.forEach((banner, i) => {
      console.log(`\n[轮播图 ${i + 1}]`);
      console.log('  ID:', banner.id);
      console.log('  图片:', banner.image);
      console.log('  链接类型:', banner.link_type);
      console.log('  链接目标:', banner.link_target);
    });
  } else {
    console.log('❌ 失败:', res.error_msg);
  }
}).catch(err => {
  console.error('❌ 异常:', err);
});
```

## 🔍 常见问题排查

### 问题 1: 控制台没有日志输出

**可能原因**：
1. 代码未编译
2. 页面未加载

**解决方案**：
1. 点击微信开发者工具的「编译」按钮
2. 确保首页已加载
3. 检查调试器是否打开

---

### 问题 2: 图片不显示

**检查步骤**：

1. **查看日志中的图片 URL**：
   ```
   [轮播图 1] image: "https://p1.mingdaoyun.cn/..."
   ```

2. **在浏览器中打开 URL**：
   - 如果能打开 → 图片正常
   - 如果打不开 → 检查 HAP 中的图片

3. **检查小程序域名配置**：
   - 登录微信小程序后台
   - 服务器域名 → downloadFile 合法域名
   - 添加 `https://p1.mingdaoyun.cn`

---

### 问题 3: 显示默认轮播图

如果看到：
```
[轮播图] ❌ 加载失败或无数据
[轮播图] 使用默认轮播图
```

**原因**：HAP 中没有符合条件的数据

**解决方案**：
1. 在 HAP 中设置 `是否启用 = 是`
2. 或者检查筛选条件是否正确
