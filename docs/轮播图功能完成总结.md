### 2. 代码实现

#### utils/api.js

✅ 实现 `getBanners()` 函数：
- 使用字段名称：`是否启用`、`排序 `、` 链接类型`、` 链接目标`
- 使用字段别名：`image` 获取图片
- 筛选条件：`是否启用 = 是`
- 排序：按 `排序` 升序

```javascript
async function getBanners() {
  const result = await getRows(WORKSHEET_ID.banners, {
    filter: { 
      type: 'group', 
      logic: 'AND', 
      children: [{ 
        type: 'condition', 
        field: '是否启用',
        operator: 'eq', 
        value: ['是'] 
      }] 
    },
    pageSize: 10,
    sorts: [{ field: '排序', isAsc: true }]
  });

  if (result.success && result.data?.rows) {
    return { success: true, data: result.data.rows.map(row => ({
      id: row.rowid,
      image: row.image?.[0]?.downloadUrl || row.image?.[0]?.url || '',
      link_type: formatOption(row['链接类型']),
      link_target: row['链接目标'] || ''
    })) };
  }
  return { success: false, data: [] };
}
```

async loadBanners() {
  const res = await api.getBanners();
  if (res.success && res.data.length > 0) {

  } else {
    this.setData({
      banners: [{ image: 'https://fp1.mingdaoyun.cn/customIcon/0_lego.svg' }]

  }
}

## ❓ 常见问题

### Q: 轮播图不显示？
**A**: 
1. 检查 HAP 中是否有 `是否启用=是` 的记录
2. 运行 `api.getBanners()` 查看返回
3. 查看控制台报错

### Q: 图片加载失败？
**A**:
1. 检查图片 URL 是否可访问
2. 确认小程序后台配置了域名
3. 检查图片格式（建议 JPG/PNG）

### Q: 如何添加更多轮播图？
**A**:
1. 在 HAP 中新建记录
2. 上传图片
3. 设置 `是否启用=是`
4. 设置 `排序` 值
5. 保存即可
