# 友趣小程序登录逻辑说明

## 登录流程概览

```
用户触发登录
    ↓
微信 wx.login() 获取 code
    ↓
调用后端 API code2session (换取 openid)
    ↓
同步用户到 HAP 用户表 (查找或创建)
    ↓
合并用户信息，存储到本地
    ↓
登录完成
```

## 详细流程说明

### 1. 检查登录状态 (checkLogin)

**触发时机**: 小程序启动时 (onLaunch)

**代码位置**: `app.js` 第 14-21 行

```javascript
checkLogin() {
  const userInfo = wx.getStorageSync('userInfo');
  if (userInfo && userInfo.hapUserId) {
    this.globalData.userInfo = userInfo;
    this.globalData.openid = userInfo.openid;
  }
}
```

**逻辑**:
- 从本地存储读取 userInfo
- 如果有 userInfo 且包含 hapUserId，则恢复登录状态
- 否则用户未登录，等待用户主动触发登录

---

### 2. 微信登录 (login)

**触发时机**: 用户点击"登录"按钮或需要登录的操作

**代码位置**: `app.js` 第 24-95 行

**步骤详解**:

#### Step 1: 获取微信 code
```javascript
wx.login({
  success: (res) => {
    // res.code 是微信临时登录凭证
  }
});
```

#### Step 2: code 换取 openid
调用后端 API `code2session`:
- URL: `https://100000whys.cn/youqu/api/user/code2session`
- 参数: `{ code: 微信code }`
- 返回: `{ openid, nickname, avatar, token }`

**代码位置**: `utils/api.js` 第 682-715 行

```javascript
async function code2session(code) {
  return new Promise((resolve) => {
    wx.request({
      url: `${AUTH_API_BASE}/code2session`,
      method: 'POST',
      data: { code },
      success: (res) => {
        if (res.data && res.data.code === 200 && res.data.data) {
          resolve({
            success: true,
            data: {
              id: res.data.data.user.id,
              openid: res.data.data.user.openid,
              nickname: res.data.data.user.nickname,
              avatar: res.data.data.user.avatar,
              token: res.data.data.token
            }
          });
        }
      }
    });
  });
}
```

#### Step 3: 同步用户到 HAP
通过 openid 在 HAP 用户表中查找或创建用户记录:

**代码位置**: `utils/api.js` 第 730-799 行

```javascript
async function getOrCreateUser(openid, userInfo = {}) {
  // 1. 通过 openid 查找用户
  const result = await getRows(WORKSHEET_ID.users, {
    filter: { 
      type: 'group', 
      logic: 'AND', 
      children: [{ 
        type: 'condition', 
        field: '69b5db58a606df9d8178a03e',  // 微信ID字段
        operator: 'eq', 
        value: [openid] 
      }] 
    }
  });

  // 2. 如果找到，返回用户信息
  if (result.success && result.data?.rows?.length > 0) {
    return { success: true, data: { id, openid, nickname, avatar } };
  }

  // 3. 如果没找到，创建新用户
  return createRow(WORKSHEET_ID.users, {
    '69b5db58a606df9d8178a03e': openid,
    '69b5db58a606df9d8178a040': nickname,
    '69b5db58a606df9d8178a041': avatar,
    '69b5db58a606df9d8178a045': new Date().toISOString()
  });
}
```

#### Step 4: 合并用户信息并存储
```javascript
const userInfo = {
  id: hapUserRes.data.id,        // HAP 用户记录 ID
  hapUserId: hapUserRes.data.id, // HAP 用户 ID
  openid: openid,
  nickname: hapUserRes.data.nickname,
  avatar: hapUserRes.data.avatar,
  token: token
};

// 存储到全局和本地
this.globalData.userInfo = userInfo;
wx.setStorageSync('userInfo', userInfo);
wx.setStorageSync('token', token);
```

### 4. 退出登录 (logout)

**触发时机**: 用户点击"退出登录"

**代码位置**: `app.js` 第 137-142 行

```javascript
logout() {
  this.globalData.userInfo = null;
  this.globalData.openid = null;
  wx.removeStorageSync('userInfo');
  wx.removeStorageSync('token');
}
```

## 常见问题

### Q: 修改头像后下次登录头像没变？

**原因**: 修改头像时使用的是微信临时路径，只在当前会话有效

**解决方案**: 需要先上传到服务器获取永久 URL

详见: `修复说明 - 图片路径.md`

### Q: 为什么需要两套 ID？

- **openid**: 微信平台的用户标识，用于微信登录
- **hapUserId**: HAP 平台的用户记录 ID，用于业务数据关联
