## 🔧 API 调用详解

### getBanners() 函数

```javascript
// utils/api.js
async function getBanners() {
  const result = await getRows(WORKSHEET_ID.banners, {
    filter: { 
      type: 'group', 
      logic: 'AND', 
      children: [{ 
        type: 'condition', 
        field: 'is_active', 
        operator: 'eq', 
        value: ['是'] 
      }] 
    },
    pageSize: 10,
    sorts: [{ field: 'sort_order', isAsc: true }]
  });
  
  if (result.success && result.data?.rows) {
    return { 
      success: true, 
      data: result.data.rows.map(row => ({
        id: row.rowid,
        image: row.image?.[0]?.downloadUrl || row.image?.[0]?.url || '',
        link_type: formatOption(row.link_type),
        link_target: row.link_target || ''
      })) 
    };
  }
  return { success: false, data: [] };
}
```

## 🎯 轮播图点击逻辑

### onBannerTap() 事件处理

```javascript
// pages/index/index.js
onBannerTap(e) {
  const item = e.currentTarget.dataset.item;
  const linkType = item.link_type;
  const linkTarget = item.link_target;

  switch (linkType) {
    case '内容':
      // 跳转到内容详情
      if (linkTarget) {
        wx.navigateTo({
          url: `/pages/post/post?id=${linkTarget}`
        });
      }
      break;

    case '网页':
      // 打开外部网页（需要 webview 页面）
      if (linkTarget) {
        wx.navigateTo({
          url: `webview?url=${encodeURIComponent(linkTarget)}`
        });
      }
      break;

    default:
      // 无链接或未知类型
      wx.showToast({
        title: '轮播图',
        icon: 'none'
      });
      break;
  }
}
```

## ⚠️ 注意事项

### 1. 图片尺寸建议
- 推荐尺寸：**750px × 360px**
- 宽高比：**2.08:1**
- 格式：JPG/PNG
- 大小：< 500KB

### 2. 链接类型说明

| 类型 | 说明 | link_target 格式 |
|------|------|-----------------|
| 无 | 点击无跳转 | 留空 |
| 内容 | 跳转到帖子详情 | 帖子 rowid |
| 网页 | 打开外部网页 | 完整 URL |

### 3. 权限配置
确保 HAP-Appkey 和 HAP-Sign 正确配置在：
- `utils/api.js` 的 `HAP_CONFIG`
- `.mcp.json`（用于 MCP 工具）

### 4. 调试技巧

**如果轮播图不显示：**
1. 检查 HAP 中是否有启用的记录
2. 检查图片 URL 是否可访问
3. 查看微信开发者工具控制台的 network 请求

**如果点击无反应：**
1. 检查 `data-item` 绑定是否正确
2. 查看 `onBannerTap` 事件是否触发
3. 确认 `link_type` 值匹配（注意大小写）

## 🚀 扩展功能建议

### 1. 添加 webview 页面

创建 `pages/webview/webview.wxml`：
```xml
<view class="webview-container">
  <web-view src="{{url}}"></web-view>
</view>
```

创建 `pages/webview/webview.js`：
```javascript
Page({
  data: {
    url: ''
  },
  onLoad(options) {
    this.setData({
      url: decodeURIComponent(options.url || '')
    });
  }
});
```

在 `app.json` 中注册页面：
```json
{
  "pages": [
    "pages/index/index",
    "pages/webview/webview",
    ...
  ]
}
```
