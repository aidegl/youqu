## 🔍 常见问题排查

### 问题 1: 轮播图不显示

**可能原因:**
1. HAP 中没有启用的记录
2. 字段 ID 不匹配
3. 图片 URL 无法访问

**排查步骤:**
```javascript
// 1. 检查 API 返回
const api = require('./utils/api.js');
api.getBanners().then(console.log);

// 2. 检查图片 URL
const banners = getPage().data.banners;
banners.forEach(b => {
  console.log('图片 URL:', b.image);
  // 在浏览器中打开 URL 确认是否可访问
});
```

### 问题 2: 点击无反应

**可能原因:**
1. `data-item` 绑定不正确
2. `bindtap` 事件未绑定

**排查步骤:**
1. 检查 `index.wxml`:
   ```xml
   <swiper-item wx:for="{{banners}}" wx:key="id" data-item="{{item}}">
   ```
2. 检查 `index.js`:
   ```javascript
   onBannerTap(e) {
     console.log('点击事件触发', e);
     // ...
   }
   ```

### 问题 3: 字段 ID 错误

如果 API 返回空数组，检查字段 ID 是否正确：

**当前使用的字段 ID:**
```javascript
{
  image: '69c4eb1c867350d552faf075',      // 图片
  link_type: '69b5dc145c2066509f21fd24',  // 链接类型
  link_target: '69b5dc145c2066509f21fd25',// 链接目标
  is_active: '69b5dc145c2066509f21fd27',  // 是否启用
  sort_order: '69b5dc145c2066509f21fd26'  // 排序
}
```

**验证方法:**
用 MCP 工具查询工作表结构确认字段 ID。
