### 2. 代码实现

#### utils/api.js - getBanners() 函数

```javascript
async function getBanners() {
  const result = await getRows(WORKSHEET_ID.banners, {
    filter: { 
      type: 'group', 
      logic: 'AND', 
      children: [{ 
        type: 'condition', 
        field: '是否启用',
        operator: 'eq', 
        value: ['是'] 
      }] 
    },
    pageSize: 10,
    sorts: [{ field: '排序', isAsc: true }]
  });

  if (result.success && result.data?.rows) {
    return { success: true, data: result.data.rows.map(row => ({
      id: row.rowid,
      image: row.image?.[0]?.downloadUrl || row.image?.[0]?.url || '',
      link_type: formatOption(row['链接类型']),
      link_target: row['链接目标'] || ''
    })) };
  }
  return { success: false, data: [] };
}
```

**关键点**:
- ✅ 使用字段名称 `是否启用`、`排序`、`链接类型`、`链接目标`
- ✅ 使用字段别名 `image` 获取图片
- ✅ 筛选条件：`是否启用 = 是`
- ✅ 排序：按 `排序` 升序

```xml
  <swiper-item wx:for="{{banners}}" wx:key="id">
    <image class="banner-image" src="{{item.image}}" mode="aspectFill"></image>

```

#### pages/index/index.js - 数据加载

```javascript
Page({
  data: {
    banners: []
  },

  onLoad() {
    this.loadBanners();
  },

  async loadBanners() {
    const res = await api.getBanners();
    if (res.success && res.data.length > 0) {
      this.setData({ banners: res.data });
    } else {
      // 使用默认轮播图
      this.setData({
        banners: [
          { image: 'https://fp1.mingdaoyun.cn/customIcon/0_lego.svg' }
        ]
      });
    }
  }
});
```

## ❓ 常见问题

### Q1: 轮播图不显示？

**排查步骤**:
1. 运行 `api.getBanners()` 检查 API 返回
2. 检查 HAP 中是否有 `是否启用=是` 的记录
3. 查看控制台报错信息

### Q2: 图片加载失败？

**排查步骤**:
1. 检查图片 URL 是否可访问（在浏览器中打开）
2. 确认图片格式为 JPG/PNG
3. 检查微信小程序后台域名配置

### Q3: 字段名称不匹配？

**解决方案**:
使用 MCP 工具查询工作表结构：
```bash
claude "使用 hap-mcp-youqu 获取工作表 69b5dc147e3c8fc03e16dcab 的字段结构"
```
