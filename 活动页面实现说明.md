## 📊 数据结构

### 活动数据格式

```javascript
{
  id: "xxx",
  title: "🎨 春季摄影大赛",
  content: "活动描述...",
  category: "活动",
  task_type: "摄影",
  bounty_amount: 500,
  images: ["https://..."],
  likes_count: 10,
  comments_count: 5,
  views_count: 100
}
```

### 筛选逻辑

```javascript
// 按 task_type 字段筛选
filterByType(activities, type) {
  if (!type) return activities;
  return activities.filter(item => item.task_type === type);
}
```
