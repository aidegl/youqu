1. 微信开发者工具 → **清缓存** → **清除全部**
2. 点击 **编译**
3. 打开 **调试器** → **Console**

### 步骤 4: 检查小程序域名配置

如果图片 URL 有效但不显示，检查域名配置：

1. 登录 [微信小程序后台](https://mp.weixin.qq.com/)
2. **开发** → **开发管理** → **开发设置**
3. **服务器域名** → **downloadFile 合法域名**
4. 添加：`https://p1.mingdaoyun.cn`

**注意**: 
- 开发版/体验版可以在详情中勾选"不校验合法域名"
- 正式版必须配置域名

---

### 步骤 5: 检查 WXML 绑定

查看 `pages/index/index.wxml`:

```xml
<swiper class="banner-swiper" indicator-dots autoplay circular>
  <swiper-item wx:for="{{banners}}" wx:key="id">
    <image class="banner-image" src="{{item.image}}" mode="aspectFill"></image>
  </swiper-item>
</swiper>
```

**检查点**:
- [ ] `wx:for="{{banners}}"` - 数据绑定正确
- [ ] `wx:key="id"` - key 正确
- [ ] `src="{{item.image}}"` - 图片路径绑定正确

## 🐛 常见问题

### 问题 1: 图片 URL 为空

**日志显示**:
```
图片 URL: 
URL 长度：0
```

**原因**: API 未正确获取图片

**解决**:
1. 检查 `utils/api.js` 中的 `getBanners()` 函数
2. 确认使用 `row.image?.[0]?.downloadUrl`

---

### 问题 2: 图片 URL 格式错误

**日志显示**:
```
图片 URL: [object Object]
```

**原因**: 未正确提取 downloadUrl

**解决**:
```javascript
// ❌ 错误
image: row.image

// ✅ 正确
image: row.image?.[0]?.downloadUrl
```

---

### 问题 3: 域名未配置

**日志显示**:
```
图片访问测试：❌ 失败 (request:fail url not in domain list)
```

**解决**:
1. 开发版：勾选"不校验合法域名"
2. 正式版：在小程序后台配置域名

---

### 问题 4: banners 数量为 0

**日志显示**:
```
[轮播图] 当前页面 banners 数量：0
```

**原因**: setData 未生效

**解决**:
1. 检查 `this.setData({ banners: res.data })` 是否执行
2. 检查 WXML 中的 `{{banners}}` 绑定
