## 📋 配置内容

### 1. 颜色配置

**轮播图指示点颜色**:
- **选中**: `#1296db`（蓝色）
- **非选中**: `#cdcdcd`（灰色）

**文件位置**: `config/colors.js`

```javascript
module.exports = {
  banner: {
    active: '#1296db',      // 选中颜色
    inactive: '#cdcdcd'     // 非选中颜色
  }
};
```

### 2. 样式配置

**文件位置**: `pages/index/index.wxss`

```wxss
/* 轮播图指示点颜色 */
.banner-swiper .swiper-pagination-bullet {
  background-color: #cdcdcd;  /* 非选中 */
}

.banner-swiper .swiper-pagination-bullet-active {
  background-color: #1296db;  /* 选中 */
}
```

## 📥 下载图标

### 方法 1: 在微信开发者工具中运行脚本

1. 打开微信开发者工具
2. 打开调试器 → Console
3. 复制并运行 `download-icons.js` 的内容

```javascript
const api = require('./utils/api.js');
// 运行 download-icons.js 中的代码
```

### 方法 2: 手动下载

1. 打开 HAP 轮播图工作表
2. 点击图片字段的下载链接
3. 保存到 `images/` 文件夹

## 🎨 颜色使用规范

### 轮播图指示点

| 状态 | 颜色 | 使用场景 |
|------|------|---------|
| 选中 | `#1296db` | 当前显示的轮播图 |
| 非选中 | `#cdcdcd` | 其他轮播图 |

### 扩展颜色（可选）

```javascript
// config/colors.js
module.exports = {
  // 主题色
  primary: '#1296db',
  
  // 辅助色
  secondary: '#cdcdcd',
  
  // 文本颜色
  text: {
    primary: '#333333',
    secondary: '#666666',
    disabled: '#999999'
  }
};
```

## 📝 注意事项

### 1. 图标文件夹

如果 `images/` 文件夹不存在，需要先创建：
```
D:\Item\youqu\images\
```

### 2. 图标命名建议

```
banner-1.png  ← 轮播图 1
banner-2.png  ← 轮播图 2
banner-3.png  ← 轮播图 3
banner-4.png  ← 轮播图 4
```

### 3. 图标尺寸建议

- **推荐尺寸**: 750px × 360px
- **宽高比**: 2.08:1
- **格式**: PNG
- **大小**: < 500KB
