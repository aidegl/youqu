## 2. 技术选型

- **前端**: 原生微信小程序

- **登录验证**: 宝塔进程（需要用户帮忙）

## 4. 功能模块

| 模块 | 功能 |
|-----|------|
| **首页** | 轮播图、内容列表（赏金任务/问答/吐槽分类） |
| **内容详情** | 查看内容、评论功能（参考抖音/小红书） |
| **发布** | 发布赏金任务、问答、吐槽 |
| **个人中心** | 我的发布、我的消息、设置 |
| **用户空间** | 查看他人个人空间（发布的悬赏任务、问答、吐槽） |
| **消息** | 系统消息、评论通知、点赞通知、**任务消息** |
| **任务** | 一对多/一对一任务接取与交付 |

## 5. 消息类型说明

- **系统消息**: 平台公告、系统通知
- **评论消息**: 被评论通知
- **点赞消息**: 被点赞通知
- **任务消息**: 每个悬赏任务有对应的任务消息（包括接取、完成、验收等）

## 6. 任务类型说明

- **一对多**: 一个人发布，多个人接取，采纳一个最优答案
- **一对一**: 一个人发布，一个人接取，私密对接

## 7. 用户个人空间

访问路径：`/pages/profile-space/profile-space?id=用户ID`

显示内容：
- 用户基本信息（头像、昵称）
- **发布的悬赏任务**列表
- **发布的问答**列表
- **发布的吐槽**列表
