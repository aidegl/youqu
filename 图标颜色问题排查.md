# 🎨 底部菜单图标颜色问题排查

**问题**: 文字颜色变了，但图标颜色没变

---

## 🔍 原因分析

### 可能原因 1: 图标本身有颜色

如果 `home_fill.png` 等图标本身已经是彩色（不是纯黑色/灰色），那么 `selectedColor` 不会改变图标颜色。

**微信 tabBar 图标颜色规则**:
- `selectedColor` **只对纯黑色图标有效**
- 如果图标本身有颜色，微信会保持原色显示

### 可能原因 2: 缓存问题

微信开发者工具缓存了旧图标

---

## ✅ 解决方案

### 方案 1: 使用纯黑色图标（推荐）

**步骤**:
1. 准备 4 个**纯黑色**图标（`#000000`）
2. 重命名为 `xxx_fill.png`
3. 微信会自动用 `selectedColor` 渲染

**图标要求**:
- 格式：PNG（透明背景）
- 颜色：纯黑色（`#000000`）
- 尺寸：81px × 81px

**效果**:
- 非选中：自动变灰色（`#cdcdcd`）
- 选中：自动变紫色（`#3054EB`）

### 方案 2: 准备两套彩色图标

**步骤**:
1. 非选中图标：灰色版本
2. 选中图标：紫色版本（`#3054EB`）

**示例**:
```
images/
├── home.png           ← 灰色房子
├── home_fill.png      ← 紫色房子（#3054EB）
├── news_hot.png       ← 灰色发布
├── news_hot_fill.png  ← 紫色发布（#3054EB）
├── mark.png           ← 灰色消息
├── mark_fill.png      ← 紫色消息（#3054EB）
├── people.png         ← 灰色我的
└── people_fill.png    ← 紫色我的（#3054EB）
```

---

## 🔧 快速修复

### 方法 1: 清除缓存

1. 微信开发者工具 → **清缓存** → **清除全部**
2. **重新编译**
3. 如果还不行，**重启微信开发者工具**

### 方法 2: 检查图标颜色

在图片查看器中打开 `home_fill.png`，检查颜色：
- 如果是**彩色** → 需要改成纯黑色或紫色
- 如果是**纯黑色** → 清除缓存后应该生效

### 方法 3: 使用在线工具转换

推荐工具：
- [Photopea](https://www.photopea.com/) - 在线 PS
- [Remove.bg](https://www.remove.bg/) - 去背景
- [IconFont](https://www.iconfont.cn/) - 下载单色图标

---

## 📝 当前配置

```json
"tabBar": {
  "color": "#cdcdcd",       // 文字非选中颜色
  "selectedColor": "#3054EB", // 文字和图标选中颜色
  "list": [
    {
      "iconPath": "images/home.png",
      "selectedIconPath": "images/home_fill.png"
    }
  ]
}
```

**注意**: `selectedColor` 会同时影响**文字**和**图标**，但前提是图标是纯黑色的！

---

## ✅ 验证方法

### 测试 1: 检查图标文件

在控制台运行：
```javascript
const fs = wx.getFileSystemManager();
const info = fs.getFileInfo({
  filePath: 'images/home_fill.png',
  success: (res) => {
    console.log('文件大小:', res.size, 'bytes');
    console.log('如果文件很小（<1KB），可能是纯色图标');
  }
});
```

### 测试 2: 临时更换图标

临时把 `home_fill.png` 换成一个纯紫色图标，看是否显示紫色：
- 如果显示紫色 → 原图标有问题
- 如果还是灰色 → 缓存或配置问题

---

**建议**: 如果图标本身是彩色的，直接使用方案 2（准备两套彩色图标）最简单！
