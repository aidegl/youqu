## 🧪 如何调试

### 步骤 1: 打开调试器

1. 打开微信开发者工具
2. 编译项目
3. 点击顶部菜单「调试器」
4. 选择「Console」面板

### 步骤 2: 查看日志

页面加载时会自动输出：

```
=== [轮播图] 开始加载 ===
[轮播图] 调用 API...
[API.getBanners] 开始请求 HAP 轮播图数据...
[API.getBanners] 工作表 ID: 69b5dc147e3c8fc03e16dcab
[API.getBanners] 筛选条件：是否启用=是
[API.getBanners] 排序：排序升序
[API.getBanners] HAP 响应：{success: true, data: {...}}
[API.getBanners] success: true
[API.getBanners] rows 数量：2
[轮播图] API 响应：{success: true, data: Array(2)}
[轮播图] success: true
[轮播图] data 长度：2
[轮播图] ✅ 加载成功，数据：[...]
[轮播图 1] {id: "...", image: "https://...", ...}
[轮播图 2] {id: "...", image: "https://...", ...}
=== [轮播图] 加载结束 ===
```

### 步骤 3: 分析结果

**✅ 成功标志**：
- `[API.getBanners] success: true`
- `[API.getBanners] rows 数量：2` (或更多)
- `[轮播图] ✅ 加载成功`

**❌ 失败标志**：
- `[API.getBanners] rows 数量：0` → HAP 中无数据
- `[API.getBanners] success: false` → API 调用失败
- `[轮播图] ❌ 加载失败或无数据` → 使用默认轮播图
