# 📱 底部菜单（TabBar）配置说明

**日期**: 2026-03-26

---

## 🎨 颜色配置

根据轮播图指示点颜色：
- **选中**: `#1296db`（蓝色）
- **非选中**: `#cdcdcd`（灰色）

---

## 📋 配置步骤

### 步骤 1: 准备图标

需要为每个 tab 准备 2 个图标（选中/非选中）：

```
images/
├── home.png          ← 首页（非选中）
├── home-active.png   ← 首页（选中）
├── publish.png       ← 发布（非选中）
├── publish-active.png ← 发布（选中）
├── message.png       ← 消息（非选中）
├── message-active.png ← 消息（选中）
├── profile.png       ← 我的（非选中）
└── profile-active.png ← 我的（选中）
```

**图标要求**:
- 尺寸：81px × 81px
- 格式：PNG
- 背景：透明

### 步骤 2: 更新 app.json

```json
{
  "tabBar": {
    "color": "#cdcdcd",        // 非选中颜色（灰色）
    "selectedColor": "#1296db", // 选中颜色（蓝色）
    "backgroundColor": "#FFFFFF",
    "borderStyle": "black",
    "list": [
      {
        "pagePath": "pages/index/index",
        "text": "首页",
        "iconPath": "images/home.png",
        "selectedIconPath": "images/home-active.png"
      },
      {
        "pagePath": "pages/publish/publish",
        "text": "发布",
        "iconPath": "images/publish.png",
        "selectedIconPath": "images/publish-active.png"
      },
      {
        "pagePath": "pages/messages/messages",
        "text": "消息",
        "iconPath": "images/message.png",
        "selectedIconPath": "images/message-active.png"
      },
      {
        "pagePath": "pages/profile/profile",
        "text": "我的",
        "iconPath": "images/profile.png",
        "selectedIconPath": "images/profile-active.png"
      }
    ]
  }
}
```

---

## 🎨 图标设计建议

### 首页
- 非选中：灰色房子图标
- 选中：蓝色房子图标

### 发布
- 非选中：灰色加号图标
- 选中：蓝色加号图标

### 消息
- 非选中：灰色对话气泡图标
- 选中：蓝色对话气泡图标

### 我的
- 非选中：灰色用户图标
- 选中：蓝色用户图标

---

## 📥 获取图标

### 方法 1: 使用图标库

推荐图标库：
- [IconFont](https://www.iconfont.cn/)
- [Flaticon](https://www.flaticon.com/)
- [Icons8](https://icons8.com/)

### 方法 2: 从 HAP 下载

如果 HAP 中有图标资源，可以运行：

```javascript
const api = require('./utils/api.js');
// 运行 download-icons.js
```

### 方法 3: 使用默认图标

微信开发者工具提供默认图标：
1. 右键 `images/` 文件夹
2. 新建 → 图片
3. 使用默认图标
