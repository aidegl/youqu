## 🐛 常见问题排查

### 问题 1: 活动数量为 0

**日志显示**:
```
[API.getPosts] rows 数量：0
```

**原因**: HAP 中没有 `category='活动'` 的数据

**解决**:
1. 运行 `add-activity-data.js` 添加活动数据
2. 或在 HAP 中手动创建活动记录

---

### 问题 2: API 调用失败

**日志显示**:
```
[API.getPosts] API 响应：{success: false, error_msg: "..."}
```

**原因**: HAP 配置错误或网络问题

**解决**:
1. 检查 `utils/api.js` 中的 `HAP_CONFIG`
2. 检查网络连接
3. 查看 HAP 应用权限

---

### 问题 3: setData 后数据仍为空

**日志显示**:
```
[活动页面] data 长度：5
[活动页面] ✅ setData 完成
[活动页面] 当前 activities: 0  ← 应该是 5！
```

**原因**: setData 未生效

**解决**:
1. 检查是否有异步问题
2. 在 setData 回调中检查数据
3. 重启微信开发者工具

---

### 问题 4: 页面白屏

**原因**: 页面路由未配置

**解决**:
检查 `app.json` 是否包含：
```json
"pages": [
  ...,
  "pages/activities/activities"
]
```

## 🚀 快速诊断命令

在控制台运行：

```javascript
// 1. 检查页面是否加载
const pages = getCurrentPages();
console.log('当前页面:', pages[pages.length - 1].route);

// 2. 检查活动数据
const api = require('./utils/api.js');
api.getPosts('活动', 1, 10).then(res => {
  console.log('活动数据:', res.data);
  console.log('活动数量:', res.data?.length);
});

// 3. 检查所有帖子（确认 API 正常）
api.getPosts('', 1, 10).then(res => {
  console.log('所有帖子:', res.data?.length);
});
```
