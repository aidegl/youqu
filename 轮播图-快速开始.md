# 🎠 轮播图 - 5 分钟快速开始

## 第 1 步：启用 HAP 中的轮播图（2 分钟）

### 方法 A: 手动更新（推荐新手）

1. 打开明道云 HAP
2. 进入「轮播图」工作表
3. 编辑「广告 1」记录
4. 设置 **是否启用 = 是**
5. 设置 **排序 = 1**
6. 保存

对「广告 2」重复上述步骤（排序=2）

### 方法 B: 运行脚本（推荐开发者）

在微信开发者工具控制台运行：
```javascript
const api = require('./utils/api.js');
// 运行 enable-banners.js 的内容
```

---

## 第 2 步：测试 API（1 分钟）

在微信开发者工具控制台运行：
```javascript
const api = require('./utils/api.js');
api.getBanners().then(res => {
  if (res.success) {
    console.log('✅ 成功！共', res.data.length, '条轮播图');
    res.data.forEach(b => console.log('  -', b.image.substring(0, 50) + '...'));
  } else {
    console.log('❌ 失败:', res.error_msg);
  }
});
```

**预期输出**:
```
✅ 成功！共 2 条轮播图
  - https://p1.mingdaoyun.cn/.../xxx.png...
  - https://p1.mingdaoyun.cn/.../yyy.png...
```

---

## 第 3 步：查看效果（1 分钟）

1. 微信开发者工具 → 编译
2. 查看模拟器首页
3. 确认轮播图显示

---

## ❓ 遇到问题？

### 轮播图不显示？
```javascript
// 检查 API 返回
const api = require('./utils/api.js');
api.getBanners().then(console.log);
```

### 图片加载失败？
- 检查 HAP 中图片是否上传
- 在浏览器中打开图片 URL 测试
- 确认小程序后台配置了域名

查看 `docs/轮播图实现说明.md`
